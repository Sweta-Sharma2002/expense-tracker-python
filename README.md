# Expense Tracker (Python)

## Overview
This project is a lightweight expense tracking system developed using Python. 
It allows users to record daily expenses, categorize transactions, and generate summary insights.

## Features
- Add expense records (date, category, amount, description)
- Persistent storage using CSV
- Category-wise expense aggregation
- Total expense calculation
- Graphical visualization using Matplotlib

## Technologies Used
- Python
- Pandas
- Matplotlib
- CSV file handling

## Future Improvements
- Database integration (MySQL)
- GUI or Web Interface
- Multi-user authentication system
